# Kentucky News Domination Roadmap (90-Day Plan)

## Objective

Become the default homepage for Kentucky local news by building
structured, hyperlocal infrastructure that Google News does not provide.

------------------------------------------------------------------------

# Phase 1 (Days 1--30)

## Build the Foundation

### Goals

-   Establish county authority pages for all 120 Kentucky counties
-   Implement precise county tagging
-   Standardize AI summaries
-   Complete a comprehensive Kentucky source database

### 1. County Authority Pages

Create structured pages:

/kentucky/\
/kentucky/{county-name}-county-news

Each page should include: - County overview (unique text) - Population -
County seat - Major cities - School districts - Live filtered news
feed - Source list

Purpose: - Long-tail SEO traffic - Geographic authority - Structured
discoverability

------------------------------------------------------------------------

### 2. County Tagging System

Each article must include: - primary_county - related_counties - state -
category

Use: - Gazetteer matching - School district references - Sheriff
department names - Circuit court references

Accuracy is critical.

------------------------------------------------------------------------

### 3. AI Summary Standard

Summary rules: - 55--65% of original length - Neutral tone - Bullet key
facts first - Clear separation of summary and context

Standard format:

Summary\
Key Points\
Full Context

------------------------------------------------------------------------

### 4. Master Source Database

Build a verified list of: - Newspapers - TV stations - School
districts - Sheriff departments - County governments - Community
bulletins - Official Facebook pages

This becomes your long-term moat.

------------------------------------------------------------------------

# Phase 2 (Days 31--60)

## Differentiate from Google News

### Goals

-   Personalization
-   School tracking
-   Public safety aggregation
-   SEO hardening

------------------------------------------------------------------------

### 1. "Only My County" Mode

Allow users to: - Select a county - Save preference - Default homepage
to their county

------------------------------------------------------------------------

### 2. Kentucky School Tracker

Pages: /kentucky/schools\
/kentucky/{county}/schools

Aggregate: - District websites - Board agendas - Closures - Budget
announcements

------------------------------------------------------------------------

### 3. Public Safety Hub

Page: /kentucky/public-safety

Aggregate: - Sheriff releases - State police updates - Arrest logs -
Missing persons - Amber alerts

------------------------------------------------------------------------

### 4. Election Mode

Prepare infrastructure for: - County clerks - Local offices - Candidate
filings - Ballot measures

------------------------------------------------------------------------

### 5. SEO Hardening

Implement: - Schema.org NewsArticle - Breadcrumb schema - Static
generation - Cloudflare edge caching - XML sitemap per county

Target: Sub-1 second load time.

------------------------------------------------------------------------

# Phase 3 (Days 61--90)

## Defensibility + Community Lock-In

### Goals

-   Build dashboards
-   Introduce monetization
-   Launch email digests
-   Enable community discussion

------------------------------------------------------------------------

### 1. County Dashboards

Each county page includes: - Recent news - Crime count (7 days) - School
updates - Government meetings - Weather - Upcoming events

------------------------------------------------------------------------

### 2. Local Business Spotlight

Offer: - Sponsored listings - Rotating county business highlights -
Non-intrusive placements

------------------------------------------------------------------------

### 3. Weekly Email Digest (Per County)

Auto-generate: - Top 10 stories - School updates - Public safety
highlights

Email builds retention.

------------------------------------------------------------------------

### 4. Moderated Comment System

Allow: - Anonymous display names - AI moderation - County-based
discussion threads

------------------------------------------------------------------------

### 5. Analytics & Expansion

At Day 90: - Identify top 10 counties by traffic - Expand editorial
depth - Add manual investigative summaries - Strengthen high-performing
regions

------------------------------------------------------------------------

# Success Criteria by Day 90

You should have: - 120 optimized county pages - Accurate tagging
engine - School and safety hubs - Personalized county mode -
County-based email list - Sub-second performance - High trust neutral
summaries

------------------------------------------------------------------------

# Long-Term Vision

Position yourself as the infrastructure layer for Kentucky local news.

Once proven, replicate the system in: - Tennessee - West Virginia -
Indiana
